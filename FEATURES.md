# Themes

## Color Theme

**Nythera Nightcode**

A warm, dimmed dark theme designed for long coding sessions.

---

## What you get

- Comfortable, low-intensity color palette
- Clear syntax separation (functions, variables, keywords)
- Balanced contrast — not too bright, not too flat
- Consistent UI across editor, sidebar, and panels
- Minimal visual noise for better focus

---

## Designed for

- Long debugging sessions
- Backend and general-purpose development
- Developers who prefer calm, stable visuals

---

## Notes

- Colors are tuned for readability over long periods
- Behavior remains consistent across different languages
- Works best with default or near-default VS Code layout
