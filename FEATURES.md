# Themes

## Color Theme

**Nythera Nightcode**

A dimmed dark theme with a balanced color system and clear syntax.

---

## What you get

- Dimmed, low-intensity color palette
- Clear syntax separation (functions, variables, keywords)
- Balanced contrast — not too bright, not too flat
- Consistent UI across editor, sidebar, and panels
- Minimal visual noise

---

## Designed for

- General-purpose development
- Backend workflows
- Developers who prefer stable, consistent visuals

---

## Notes

- Colors are tuned for clarity, not intensity
- Behavior remains consistent across different languages
- Works best with default or near-default VS Code layout
