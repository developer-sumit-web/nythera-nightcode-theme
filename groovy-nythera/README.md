## ✨ Overview

**Groovy Nythera** is a dark Visual Studio Code theme that blends the deep navy UI of **Nythera** with a warm syntax palette inspired by **Gruvbox**.

The result is a comfortable developer environment designed to keep focus on the code while maintaining strong syntax readability.

This theme emphasizes:

• balanced contrast
• warm color harmony
• minimal UI distraction
• clear syntax distinction

---

## 🎨 Features

| Feature         | Description                                            |
| --------------- | ------------------------------------------------------ |
| Deep UI Palette | Navy-based interface derived from Nythera              |
| Warm Syntax     | Gruvbox-inspired color system                          |
| Eye Comfort     | Reduced harsh whites for long coding sessions          |
| Clear Syntax    | Distinct colors for functions, variables, and keywords |
| Balanced UI     | Subtle separation between editor, sidebar, and panels  |
| Multi-language  | Works consistently across many languages               |

```json
{
  "editor.cursorSmoothCaretAnimation": "on",
  "editor.cursorBlinking": "smooth",
  "editor.smoothScrolling": true,
  "editor.fontFamily": "'Fira Code', 'JetBrains Mono', Consolas, monospace",
  "editor.fontSize": 13,
  "editor.lineHeight": 23,
  "editor.cursorStyle": "block",
  "editor.bracketPairColorization.enabled": true,
  "editor.minimap.autohide": "mouseover",
  "workbench.startupEditor": "none",
  "workbench.list.smoothScrolling": true,
  "window.zoomLevel": -0.1,
  "terminal.integrated.cursorBlinking": true,
  "terminal.integrated.smoothScrolling": true,
  "workbench.colorTheme": "Groovy Nythera"
```

## 🧠 Design Philosophy

Groovy Nythera follows two main design principles.

### Minimal Interface

The UI stays subtle and calm so that the focus remains on the code.

### Warm Readability

The syntax palette is inspired by **Gruvbox**, known for its warm tones and clear visual hierarchy.
