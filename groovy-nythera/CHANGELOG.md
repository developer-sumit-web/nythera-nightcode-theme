# Changelog

All notable changes to the **Groovy Nythera** theme will be documented in this file.

The format is based on _Keep a Changelog_ and follows semantic versioning principles.

---

## [1.0.0] - 2026

### Added

- Initial release of the **Groovy Nythera** Visual Studio Code theme
- Deep navy UI based on the Nythera interface design
- Gruvbox-inspired syntax highlighting palette
- Carefully tuned semantic token colors
- Balanced contrast across the editor, sidebar, and UI elements
- Terminal ANSI color palette for consistent terminal readability

### Notes

- Groovy Nythera is a variant of the **Nythera** theme that experiments with a warmer syntax color system.
- The UI structure follows the Nythera design philosophy while adapting syntax colors inspired by Gruvbox.
